The Low-Latency HLS (LL-HLS) Plugin for Ant Media Server lets you create a Low-Latency HTTP Live Streaming endpoint for streams ingested by WebRTC, RTMP, RTSP, etc. The LL-HLS Plugin provides latency between 2-5 secs, while the traditional HLS latency is more than 10-30 secs.
